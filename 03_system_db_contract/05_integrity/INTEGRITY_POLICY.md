# Integrity Policy

- 모든 Function ID가 Registry에 존재
- 모든 Schema Path가 실제 존재
- 모든 Seed ID가 Master Index에 존재
- 모든 prior_seed·next_seed가 실제 존재
- Lineage Edge가 prior·next 연결과 일치
- Result에서 Source까지 역추적 가능
- Result에 Evidence와 Comparison이 존재
- Result에 Next Question이 존재
- Provenance의 Function·Seed·Commit 참조가 존재
- for_instance Canonical Data를 Relation이 복제하지 않음

# Active_Schema GitHub Mirror

이 Directory는 Zenodo에 게시된 HRTDB Active_Schema의 GitHub 공개 Mirror다.

```yaml
latest_version: "2.0.0"
version_doi: "10.5281/zenodo.21453793"
concept_doi: "10.5281/zenodo.21453792"
publication_zip_sha256: "0b0317dadfd333c151c9b3f5e4c930d048c76e39fe037914f01ae9767c6d2b30"
```

```text
Zenodo Exact Publication
→ 외부 고정 Root

GitHub unpacked mirror
→ 사람이 읽고 관계를 따라갈 수 있는 공개 Tree
```

이 Mirror는 Track DB 정본을 대체하지 않는다. Track DB Payload는 기존 Canonical 위치에 남는다.
